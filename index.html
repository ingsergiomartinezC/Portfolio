# Flujo de trabajo típico
$ git checkout -b feature/actualizar-banner-hero
$ # ... editar HTML/CSS ...
$ git add .
$ git commit -m "feat: actualizar banner hero con nueva imagen y CTA"
$ git push origin feature/actualizar-banner-hero
$ # Abrir Pull Request → revisión → merge a develop
